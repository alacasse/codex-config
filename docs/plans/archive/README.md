# Plan Archive

`docs/plans/archive/` is the **Plan Archive** for completed or superseded
planning documents.

Archived files preserve historical planning evidence and may still contain
paths that were correct when they were active. Treat those internal paths as
historical unless a current active document under `docs/plans/` points to them.

Runtime runner artifacts, cache files, sessions, logs, ignored local artifacts,
and `architecture-program-runs/` do not belong here.
